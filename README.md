## ğŸ¨ Customization

### Changing the Number of Images

Modify the `--quantity` CSS variable in `index.html`:

```html
<div class="slider" style="--quantity: 10">
```

### Adjusting Animation Speed

Edit the animation duration in `style.css`:

```css
animation: autoRun 20s linear infinite;
```

### Modifying 3D Perspective

Adjust the `translateZ` value to change the carousel radius:

```css
translateZ(550px);
```
